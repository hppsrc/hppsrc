Hppsrc API

This is not a real API ... kinda.

Any request of the API with "ps-" will only be available using irm and iex with powershell.

Format: /api/ps-<powershell script name>

Example: irm "https://hppsrc.vercel.app/api/ps-ping" | iex

Endpoints

  • ping: /api/ps-ping

    ❘ Get a pong

  • timer: /api/ps-timer

    ❘ Open a timer

  • installer: /api/ps-installer

    ❘ Hppsrc script installer
